# Ejercicios Básicos de HTML + CSS (Nivel Inicial)

Estos ejercicios son simples y prácticos. Cada uno se enfoca en un concepto central de CSS: **Flex**, **Grid**, **margin**, **padding**, **gap**, **border**, **border-radius**, **media queries**, y algunas propiedades básicas más.
Ir avanzando sobre los ejercicios anteriores para reutilizar el código.

## 1) Caja con margen, padding y borde

**Objetivo:** Practicar el modelo de caja.

**Requisitos:**

- Crear un `div` de 200×200 px.
- Agregar `margin`, `padding`, `border` y `border-radius`.
- Cambiar el color de fondo y del borde.

## 2) Tarjeta con varias cajas en fila usando Flex

**Objetivo:** Introducir `display: flex` y `gap`.

**Requisitos:**

- Crear un contenedor con 3 cajas iguales en una fila.
- Usar `display: flex` y `gap` para separar las cajas.
- Cada caja con `width`, `height` y un color diferente.

## 3) Galería simple con Grid

**Objetivo:** Introducir `display: grid`.

**Requisitos:**

- Crear un contenedor con 6 cajas.
- Usar `display: grid` y `grid-template-columns: repeat(3, 1fr)`.
- Usar `gap` para separación.

## 4) Layout responsive con Media Queries

**Objetivo:** Ver diseño adaptable a pantallas pequeñas.

**Requisitos:**

- Reutilizar el ejercicio de **Flex** o **Grid**.
- Agregar una `@media (max-width: 600px)` para que las cajas se acomoden en **1 sola columna** en pantallas pequeñas.

## 5) Contenedor centrado en pantalla

**Objetivo:** Combinar varias propiedades.

**Requisitos:**

- Crear un contenedor con ancho fijo (ej: 400 px).
- Centrarlo en la pantalla con `margin: auto` y un `max-width` para que no se salga del borde en móviles.
- Agregar `padding` interno y un fondo de color.

## 6) Grid con columnas flexibles

**Objetivo:** Hacer un grid adaptativo.

**Requisitos:**

- Crear un grid con `grid-template-columns: repeat(auto-fit, minmax(150px, 1fr))`.
- Incluir 8–10 cajas con fondo y texto.
- Ajustar la separación con `gap`.

## 7) Caja con borde redondeado y sombra

**Objetivo:** Practicar `border-radius` y propiedades visuales básicas.

**Requisitos:**

- Crear una caja con `border-radius` de 20 px.
- Agregar un color de fondo y una sombra (`box-shadow`).
- Definir un ancho y alto fijos.
